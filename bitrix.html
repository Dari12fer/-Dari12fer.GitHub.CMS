1С-Битрикс

1С-Битрикс — одна из популярных систем управления сайтами. Это единственная коммерческая платформа, созданная в России, которой удалось завоевать мировую популярность.

Сайты, для которых подойдет 1С-Битрикс

Платформа ориентирована на крупные коммерческие проекты. Чаще всего CMS используют для создания интернет-магазина с тысячами товарных позиций или корпоративного портала.

Лицензия начального уровня позволяет создать небольшой проект, который позже можно масштабировать. Форумы, блоги и личные сайты не имеет смысла создавать на этой платформе из-за высокой стоимости тарифных планов.

Удобство администрирования

1С-Битрикс рассчитана на опытных вебмастеров. Чтобы получить полное представление о возможности CMS, придётся пройти платные курсы.

Модули и утилиты

Практически все модули условно бесплатные, их можно скачать и установить, но для полноценного использования необходимо приобрести лицензию.

Популярные дополнения:

  • турбо-страницы Яндекс
  • интеграция СДЭК
  • создание прайс-листа
  • учёт доставки
  • эквайринг
  • онлайн-кредитование
  • система скидок
  • товар дня
  • параметры заказа
  • KPI

Системные требования

  • Веб-сервер: Apache
  • OS: UNIX, Windows
  • PHP 7.1 и выше
  • БД: MySQL 5.6 и выше
  • Поддержка протокола HTTPS

Безопасность и производительность

Именно из-за высокой степени защиты и производительности веб-мастера выбирают эту CMS для коммерческих проектов.

1С-Битрикс позволяет администратору полностью контролировать все изменения в системе через логи. Движок изначально разрабатывался с целью защитить веб-ресурс от возможных атак. Служба поддержки ищет уязвимости в системе и регулярно обновляет код платформы.

С помощью панели “Группы безопасности” веб-мастер может настроить правила авторизации, включая двухэтапную аутентификацию.

Возможности для самостоятельного конфигурирования

Можно устанавливать собственные модули или использовать предложенные в каталоге утилиты. С их помощью можно значительно расширить функциональность CMS и масштабировать свой проект.

Вокруг 1С-Битрикс образовалась профессиональное сообщество, оказывающее услуги разработки и интеграции модулей.
